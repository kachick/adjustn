adjustn

A tiny text tool for adjusting new line characters(CR LF CRLF). In short, Like unix2dos/dos2unix on Ruby.

  • CR (Mac)

  • LF (Unix)

  • CRLF (Windows)

Convertible above

Usage

Require Ruby 2.6 or later

Overview

$ gem install adjustn
$ adjustn.rb LF path [*path]
Done!
